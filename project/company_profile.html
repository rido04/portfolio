Project 4 - Company Profile Website

Here i`m built a modern company profile website using Laravel 11, with Filament handling the admin panel. The frontend is fully customized with Blade and TailwindCSS for a sleek and responsive design.

This project blends performance, aesthetics, and usability to create a great user experience.


Features:

  • Easy Content Management – Filament makes it simple to update and manage content.
  • Custom Landing Page – Enhanced with Particle.js effects and smooth typing animations for a dynamic feel.
  • Portfolio Showcase – A clean and professional way to display projects.
  • Fully Responsive – Optimized for both desktop and mobile users.
